# SymRadar
SymRadar is a patch validtion tool for conditional patches.

## Structure
### CPR
CPR is a tool for generating patches for validation.
This directory contains benchmarks and scripts for preparing, running and analyzing.


### uni-klee
UNI-KLEE (**UN**der-constra**I**ned [KLEE](https://github.com/klee/klee)) is our implementation of KLEE-based symbolic execution tool, which supports under-constrained symbolic execution and lazy initialization.
